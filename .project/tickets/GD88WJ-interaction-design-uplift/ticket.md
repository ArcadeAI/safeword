# Make safeword exceptional at interaction design

**Goal:** Hold safeword's human-facing surfaces to the same bar it holds code to: plain, self-sufficient, verified.

**Why:** Safeword's real product is the human↔agent interaction — proposing, blocking, explaining, handing off across sessions. Yet it enforces _code_ with structural gates while shaping _how it talks_ with instructions alone, the weakest tier by its own Principle 1. This epic closes that gap, starting with the highest-pain moment (gate blocks) and generalizing to a verified message contract.

## The jobs behind this (exploration)

Five talking-moments where interaction quality is make-or-break: (1) hitting a
block, (2) staying oriented mid-session, (3) coming back after a break, (4) being
asked to decide, (5) right-sized replies. 64J9R1 takes job 1; JVKMSM generalizes
the fix so the rest follow.
